Exercice 1 : Nombre de clics

  • Sélectionnez le bouton d'identifiant #clic
  • Ecoutez l'événement click
  • A chaque click inscrivez dans la console le nombre de clic réalisé
  • Indiquez dans la span #iterateur le nombre de click réalisé

Exercice 2 : Formulaire d'accueil

La propriété permettant d'accéder au contenu d'un formulaire est monInput.value.

  • Ecoutez l'événement clic sur l'envoie du bouton #sendForm
  • Selectionnez les valeurs saisies dans le formulaire aux champs #nom - #prenom - #ddn
  • Sélectionnez le contenant des résultats #resultat
  • Affichez ce contenant en enlevant la classe invisible
  • Sélectionnez puis affichez les valeurs récupérer dans les champs du formulaire dans les span d'identifiants #resNom - #resPrenom - #resAge en calculant l'âge

Exercice 3 : Carré qui bouge

Pour écouter les événements liés aux touches du clavier on va écouter l'événement keydown sur l'ensemble du document.

  • Sélecctionnez le carré noir d'identifiant sqMvt
  • A la manière de l'exemple du cours, écoutez l'événement keydown des touches haut, bas, gauche et droite afin de déplacer le carré noir
  • Lorsque le carré noir touche une des limites du container changez sa couleur en rouge (maVariable.style.backgroundColor = 'red') et empêcher la sortie du carré (quand il ne touche plus il redevient noir)

Exercice 4 : Les couleurs

Partie 1 : Aléatoire

Nombre aléatoire : Math.floor(Math.random() * 256)

  • Sélectionnez les champs du formulaire #affichRed - #affichGreen - #affichBlue
  • Ecoutez l'événement clic du bouton d'identifiant #clicColor (changer)
  • Générez 3 nombres aléatoires compris entre 0 et 255 inclus
  • Placez ces chiffres dans les différents champs du formulaire
  • Changez le style de #containerColor pour prendre en compte les 3 nombres (maVariable.style.backgroundColor = 'rgb(' + nb1 + ',' + nb2 + ',' + nb3 + ')' )

    • Partie 2 : Formulaire

    • Ecoutez l'événement clic du bouton valider d'identifiant #formColor
    • Récupérez les valeurs contenus dans les champs
    • Grâce à ces valeurs modifiez la couleur du fond de #containerColor

    Exercice 5 : Accordéon

    • Selectionnez les éléments suivants : #titre1 - #titre2 - #titre3 correspondant aux titres / #contenu1 - #contenu2 - #contenu3
